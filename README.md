## Installation 

1. - Clone the repo
 ```sh
git clone https://github.com/ggab13/To-Do-List.git
   ```

2. - Navigate to the project directory
```sh
 cd To-Do-List  
 ```
3. - Install dependencies
 ```sh
npm install
```
4. - Navigate to dist folder
 ```sh
 cd To-Do-List  
 ```
5. - Open the index.html file in your web browser to play the game.
  ```sh
 start index.html  
 ```
